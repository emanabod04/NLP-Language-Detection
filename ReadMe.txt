To Run the entire project:   
    First: Run web scrapping script Web_Scraping.py, or uncomment "# import Web_Scraping" to generate  lang.csv
    Second: You can skip first step and use the included lang.csv
    Third: Run Final_Project_NLP.ipynb Final_Project_NLP.ipynb

To Run User Interface with champion model deployed:
    First: Create flask environment
    Second: Take main.py , champion model file(Language_Detection_Module.py), Web_Scraping file, and lang.csv to save time, the put it in the environment Directory
    Third: Start environment and go to http:/127.0.0.1/
